Usage: python invoke_llm.py

A simple Python program to invoke OpenAI's free LLM (using the OpenAI API) and provide a response to user input.

Requirements:
- Python 3.7+
- openai>=1.0.0 (install with `pip install openai`)
- Set your OpenAI API key as an environment variable named `OPENAI_API_KEY`

How to run:
1. Install dependencies: `pip install -r requirements.txt`
2. Set your API key: `export OPENAI_API_KEY=your-key-here`
3. Run the program: `python invoke_llm.py`

Features:
- Interactive chatbot interface
- Type 'exit' to quit

Example:
$ python invoke_llm.py
Welcome to the OpenAI LLM chatbot! Type 'exit' to quit.
You: Hello
LLM: Hi! How can I assist you today?
You: exit
Goodbye!
